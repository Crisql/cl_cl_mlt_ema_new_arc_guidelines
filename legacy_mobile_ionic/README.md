# ClvsEMAMovil
Aplicación para manejo de rutas, gestión de documentos de marketing y socios de negocios con SAP Business One, configurable el lenguaje ingles y español

## Tipos de documentos
- Ofertas de ventas
- Órdenes de venta
- Factura deudor
- Factura deudor + Pago
- Factura de reserva
- Entregas
- Pagos recibidos
- Pagos a cuenta

## Funcionalidad Offline
Capacidad de utilizar la aplicación sin internet para creación **ofertas de venta, ordenes de venta, facturas deudor y deudor + pago**, los cuales se podrán sincronizar una vez se tenga conexión a internet
